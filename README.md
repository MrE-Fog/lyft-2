lyft.github.io

## Updating the docs

Localized strings:

* locales/en.yml

Partials for nav, search and tracking:

* source/partials

Basic and advanced docs (in markdown):

* source/basic
* source/advanced

Landing page and footer:

* source/localizable

All images:

* source/images

### Running a local middleman test server to view changes

```
bundle exec middleman server
```
